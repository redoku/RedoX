# RedoX Launcher

Кастомный лаунчер для Minecraft 1.21.1 (Fabric). Можно использовать как для игры на сервере RedoX (сервер разработчиков лаунчера), так и для обычной игры в Minecraft.

## Возможности

- Офлайн-авторизация
- Автоматическая установка Java 21 (Adoptium)
- Управление модами (Modrinth)
- Кастомные профили с выбором версии и загрузчика
- 18 тем оформления с 14 фоновыми canvas-эффектами
- Автообновление через GitHub Releases
- NSIS-установщик с выбором папки

## Требования

- Windows 10/11 (x64)
- 4 ГБ RAM
- Подключение к интернету (для первого запуска)

## Установка

1. Скачайте `RedoXSetup_v1.1.1.exe` из раздела Releases
2. Запустите установщик
3. Выберите папку для установки
4. Запустите RedoX Launcher

## Запуск из исходников

```bash
npm install
npm start
```

## Сборка установщика

```bash
npm run build
```

Результат появится в папке `dist/`.
